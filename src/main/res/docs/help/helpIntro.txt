¡Gracias por descargar 'Armame el Doparti v3.0'!

Te presento a continuación una guía muy fácil de seguir para entender cómo usar el programa y todas sus funcionalidades.

Justo abajo de este espacio de texto tenés un indicador que te muestra cuántas páginas de instrucciones hay para leer, y cuál estás leyendo.

Abajo de ese indicador tenés dos botones: 'Anterior' y 'Siguiente' que te van a servir para navegar entre las distintas páginas de ayuda, y 'Volver al menú principal', que hace exactamente lo que dice.
